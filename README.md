# push_swap
A program that sorts numbers with 2 stacks and a limited set of instructions

## Getting started
How to compile:
### `make all`

How to compile bonus part:
### `make bonus`

How to recompile:
### `make re`

How to clean:
### `make clean`

How to do a full clean:
### `make fclean`
